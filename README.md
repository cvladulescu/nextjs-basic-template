# Next.js + Tailwind CSS Example

Run this command after cloning:

```bash
yarn
```
Go to [Google Console](https://console.cloud.google.com/apis/credentials) and create your API Keys. Create a file [`.env`] in the project directory and add:

```bash
GOOGLE_SECRET="YOUR_SECRET_KEY"
GOOGLE_ID="YOUR_SECRET_ID"
```

Run this command to start the project:

```bash
yarn dev
```
